# i18n

Built-in locales:

- `en` (default)
- `fr` (basic)
- `es` (basic)

## Using a Locale

```ts
import { timeAgo } from "@apurvapawaskar/timeago-plus";

timeAgo(Date.now() - 2 * 60_000, { locale: "fr" }); // "2min plus tôt"
timeAgo(Date.now() - 2 * 60_000, { locale: "es" }); // "2min atrás"
```

Locale tags fall back to the base tag:

```ts
timeAgo(Date.now() - 2 * 60_000, { locale: "es-ES" }); // uses "es"
```

## Overriding Labels

Labels are simple strings merged over the selected locale.

```ts
import { timeAgo } from "@apurvapawaskar/timeago-plus";

timeAgo(Date.now() - 2 * 60_000, {
  labels: {
    minutes: "mins",
    pastSuffix: " ago"
  }
}); // "2mins ago"
```

See [docs/api.md](./api.md) for the full `TimeAgoLabels` shape.
